команда->опции(определяют как команда себя будет вести)->аргумент(к чему команда применяется)


Основные команды:
    pwd - вывести текущую директорию
    ls - вывести содержимое директории
    ls --all --human-readable -l путь(если путь не задан, то показывается текущая директория)
    ls -a -h -l
    ls -ahl (эквивалент)-> ls -lah
    man команда(справка по команде(выход q))
    clear (очистить экран терминала)
    exit (закрыть терминал)
    ls -l аргумент(в виде списка, размер в байтах)
    ls -hl аргумент(размер в более удобном формате)
    ls -ahl аргумент(показать скрытые папки и файлы(скрытые файлы начинаются с точки))
    Ctrl+A (перейти курсором в начало)
    Ctrl+E (перейти курсором в конец)
    n-no, y-yes, a-abotr, e-edit(отредактировать самостоятельно)
    Tab (показать возможные варианты)
    Tab+Tab (выбрать первый вариант)
    Ctrl+Shift+C (копировать в терминале)
    Ctrl+Shift+V (вставить в терминале)
    /home (путь от главной директории)
    file.txt (путь от текущей директории)
    . (тукущая директория)
    .. (директория на уровень выше)
    ~ (домашняя директория)
    * (любое количество любых символов)
    ? (ровно один любой символ)
    mkdir путь_до_директории(создать директорию)
    mkdir -p путь_до_директории(создать директорию и ее родителей)
    touch путь_до_файла(создать пустой файл)
    cd путь_до_директории(войти в директорию)
    cd (вернуться в главную директорию)
    rm путь_до_файла(удалить файл)
    rm -r путь_до_директории (удалить директорию)
    rm -rf путь_до_директории(удалить директорию без вопросов)
    cp путь_до_файла1 путь_до_файла2(скопировать файл1 в файл2)
    cp -r путь_до_дир1 путь_до_дир2(скопировать дир1 в дир2)
    mv путь1 путь2(переместить директорию или файл/директорию)
    -f (для удаления/копирования без вопросов)
    ls -la (прооверить статусы программ)
    cat имя файла (открытие содержимого файла в терминале)
    less имя_файла (открытие содержимого файла в отдельном окне)
        q (выход); / (поиск)
        g (в начало файла); G (в конец файла)
    nano имя_файла (открывает файл для редактирования)
        Ctrl+X (выход из редактирования)

Команды для работы с программами():
    firefox (открываем браузер)
    gedit (открываем блокнот)
    Ctrl+C (прервать выполнение программы)
    Ctrl+V (приостановить выполнение программы)
        fg (продолжить выполнение программы)
        bg (продолжить выполнение программамы в фоновом режиме)
    ./имя_программы (запускаем программу)
    /полный/путь/имя_программы (запускаем программу)
    /полный/путь/имя_программы & (запускаем программу сразу в фоновом режиме)
    chmod +x имя_программы (добавить программе возможность исполняться)
    имя_программы < имя_файл (брать данные из файла (stdin))
    имя_программы > имя_файл (выводить данные в файл (stdout))
    имя_программы >> имя_файл (выводить данные в файл с дозаписью (stdout))
    имя_программы > имя_файл (выводить данные об ошибках в файл (stderr))
    имя_программы >> имя_файл (выводить данные об ошибках в файл с дозаписью (stderr))
    pipe (конвеер программ)
        программа_1|программа_2|...|программа_n
        программа_1 передает stdout в качестве stdin программе_2 и т.д.
    
Команды для скачивания:    
    wget ссылка (скачать файл по ссылке и сохранить в текущую директорию(сохраниться под изначалым именем))
    wget -P путь_до_директории ссылка (скачать файл по ссылке и сохранить в директорию с заданым путем(сохраниться под изначалым именем))
    wget -O путь_до_директории/имя_файла ссылка (скачать файл по ссылке и сохранить в директорию с заданым путем(сохраниться под указаным именем))
    wget -c ссылка (докачать файл если соединение оборвалось)
    wget --spider ссылка (проверить доступность файла по ссылке)
    wget -i файл (скчачать сразу несколько файлов по ссылкам записаным в файле)
    wget --spider -i файл (можно комбинировать)
    wget -r -l глубина ссылка (скачивает все файлы на странице и переходит по всем ссылкам и также скачивает все файлы(по умолчанию на глубину 5))
    wget -r -l -A jpg,png...pdf ссылка (скачивает все файлы определенного типа)

Архивирование/Разархивирование:
    unzip архив.zip (распаковать содержимое архива.zip, не удалит исходный архив)
    gunzip архив.gz (распаковать содержимое архива.gz, удалит исходный архив)
    zip фрхив.zip файл_1 файл_2 ... файл_n (запаковывает указанные файлы в архив и присваивает указанное имя архиву)
    gzip файл (запаковать файл в архив (имя создается само и только один файл))
    tar -cvf фрхив.tar файл_1 файл_2 ... файл_n (запаковывает указанные файлы в промежуточный архив.tar без сжатия)
        -c (создание архива)
        -v (выводит информацию на экран(можно убрать, тогда на экране будет пусто, но процесс пройдет))
        -f (отвечает за то что создаваться будет именно файл в файловой системе(всегда надо указывать и именно в конце))
    tar -zcvf фрхив.tar.gz файл_1 файл_2 ... файл_n (сразу запаковывает в tar.gz)
        -z
    tar -xvf архив.tar (распаковывает архив.tar)
        -x
    tar -zxvf архив.tar.gz (распаковывает архив.tar.gz)
        -z
    gzip архив.tar (запаковывает архив.tar в архив.tar.gz, а исходный архив.tar удаляет)

    Еще один архиватор(толлько -z заменяется на -j, принцип тот же что и gzip)
        bzip2 файл (запаковать)
        buzip2 файл.bz2 (распаковать)
        tar -cjvf архив.tar.bz2 файл_1 файл_2 ... файл_n (запаковать)
        tar -xjvf архив.tar.bz2 (распаковать)
